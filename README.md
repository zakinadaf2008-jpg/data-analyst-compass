# Data Analyst Compass

An interactive dashboard designed to help aspiring data analysts track their learning journey, monitor progress, and organize courses and skills.

## Features
- Learning roadmap
- Progress tracking
- Course management
- Analytics dashboard
- User-friendly interface

## Purpose
Data Analyst Compass helps learners stay organized and motivated while developing data analysis skills.

## Tech Stack
- Built with Lovable
- Hosted on GitHub

## Future Improvements
- User authentication
- Certificates and achievements
- Quiz tracking
- Personalized learning recommendations
